Creating an algo template

To create an algo template:

  1. Click Pick Algo, and select the desired algo from the list of deployed algos.

  2. Select Manage templates from the New drop-down menu.

  3. Select Add template.

  4. In the Add template dialog, enter a Name for the template, and click Save.

    The template is added to the drop-down.

  5. Specify the desired algo parameter values, and click the icon.

Using an algo template

To use a template for an algo:

  1. Select an algo from My Algos in the Autotrader widget.

  2. Enter the number of instances of the algo template you want to instantiate.

  3. From the New drop-down menu, select the template use.

  4. Make any desired changes to the algo parameters.

  5. Click to launch the selected algo templates.

Modifying an algo template

To modify an algo template:

  1. Click Pick Algo, and select the desired algo from the list of deployed algos.

  2. Select Manage templates from the New drop-down menu.

  3. From the Template drop-down, select the template you want to modify.

  4. Update the desired algo parameter values, and click the icon.

Setting a default algo template

You can select a default template to use in Algo Dashboard.

To set a default template for an algo in Algo Dashboard:

  1. Select an algo from My Algos in the Autotrader.

  2. Select Manage templates from the New drop-down menu.

  3. Click the Templates drop-down menu.

  4. Click the icon for the template you want to make the default.

Deleting an algo template

To delete an algo template:

  1. Select an algo from My Algos in Autotrader.

  2. Select Manage templates from the New drop-down menu.

  3. Click the Templates drop-down menu.

  4. Click the Delete icon for the template you want to delete.
