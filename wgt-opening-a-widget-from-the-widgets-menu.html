Opening a widget from the Widgets menu

You can open a widget in your workspace by selecting a widget from the widget menu.

  1. Open the Widgets menu to display the available widgets.

  2. Select the widget you want to open.

    In this case, you add an MD Trader widget.

  3. Using this method, the MD Trader widget doesn't have an instrument yet. So it prompts you to set the instrument for the widget. Start searching for the desired instrument.

  4. After choosing an instrument, the widget updates with the appropriate data.

  5. Continue opening widgets to complete your workspace, such as an Order Book widget.

    The Order Book widget is not associated with an instrument or product, so you don't have to configure the widget. Order books are associated with accounts, so you can filter the display based on the Account column.

  6. Arrange the widgets so the workspace looks like you want it to look.

  7. When you are done opening and arranging widgets, save the workspace by clicking the Save icon.
