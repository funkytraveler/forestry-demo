The Autotrader display comprises the following:

  1. Grid of algo instances that lists all instances of the algo that have been loaded into Autotrader.
  2. Columns for the status of the algo instances and all of their algo variables.
  3. Tabs for each algo loaded into the Autotrader widget.
  4. Algo player controls to play, pause and delete algo instances.
  5. Row management buttons to pin, clone and delete algo instance rows.
  6. Modification controls to apply or discard variable changes for running algos.
