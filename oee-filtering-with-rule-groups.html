Creating and combining groups of filters allows you to use more complex logic for isolating trading opportunities.

Creating a filter group

To create a filter group:

  1. From the sidebar, click the icon to open the Filter builder.

  2. Click Add group.

  3. Click Add rule.

  4. Select the desired field you want to use in the filter; then select the desired criteria for the field.

  5. Click Add rule

  6. Select the desired field you want to use in the filter; then select the desired criteria for the field.

  7. In the group area, click AND or OR as desired.

  8. Click Apply.

    The contents of the Electronic Eye widget updates to show only those instruments that match the filter rule.
