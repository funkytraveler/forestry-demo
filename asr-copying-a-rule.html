To copy a rule

  1. Select a rule in the Autospreader Rule widget (e.g., Basic Slop)
  2. Click Copy.

    The copied rule appears in the Autospreader Rules list (e.g., BasicSlop - copy). You can click the edit icon to rename the copied rule.
