Filtering orders by account

To filter working orders per account in MD Trader, select and account and click the Filter check box.

Liquidating your positions

When you gain a position for an instrument, the Liquidate button is enabled, as shown.

To liquidate your position:

  1. Enable or disable the Filter check box, as desired.
  2. Click Liquidate.

Enabling EPIQ in MD Trader

  1. Click (check) the Enable Estimated Position in Queue (EPIQ) checkbox on the Preferences | Orders tab.

    Note: EPIQ is only tracked once the preference is enabled. This setting is unchecked (disabled) by default.

  2. Right-click in the MD Trader order pane to open the context menu and click Show EPIQ, and click Enable EPIQ in the dialog box that appears.

    Note: EPIQ does not consider orders placed when the Enable Estimated Position in Queue (EPIQ) setting was unchecked (disabled).

    Right-click again in the order pane and select Hide EPIQ to hide this column. If the EPIQ column is shown but the Enable Estimated Position in Queue (EPIQ) setting is unchecked (disabled), each cell in the column will be blank.

Creating a custom order type button

  1. Right-click in the MD Trader title bar and select Settings.

    The customized order type buttons are created using the Order types button option in the MD Trader settings.

  2. Click Order type buttons.
  3. Click the Order Type field to select an order type.

  4. Click the Label field to customize the button name. Check the Preview column to verify how the title will appear on the button in MD Trader.

  5. Click OK and Apply.

    The custom button appears in the MD Trader order pane. Click Save to exit settings and save the changes.
