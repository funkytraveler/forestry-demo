The RFQ Viewer display consists of the following:

  1. Pin column — {% include columns/col-pin.html %}
  2. Re-RFQ column — {% include columns/col-rerfq.html %}
  3. Time column — {% include columns/col-time-rfq.html %}
  4. Elapsed column — {% include columns/col-elapsed.html %}
  5. Contract and Type columns — Displays the options or strategies contract and instrument type for which RFQs have been submitted to the exchange.
  6. Best bid and quantity columns — Displays the best bid and quantity for the contract with a submitted RFQ.
  7. TV column — {% include columns/col-tv.html %}
  8. Best ask and quantity columns — Displays the best ask and quantity for the contract with a submitted RFQ.
  9. Volatility and Greek calculations columns — Shows implied and user-calculated volatility values. Columns for options Greeks can also be shown.
  10. Widget columns — Allow you to open MD Trader, Strategy Creation, and Blocktrader widgets seeded with data for the selected instrument. Note: Blocktrader support for CME is planned for an upcoming release.
  11. Tabs — Allow you to monitor RFQs for multiple instruments in a single RFQ Viewer.

New RFQ Notifications

When using multiple tabs in the RFQ Viewer, the tabs for the products you are not currently viewing will be highlighted yellow to indicate when new RFQs have been received. These tabs will also display the number of new RFQs received since you last viewed the products on those tabs.
