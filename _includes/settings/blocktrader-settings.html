Display settings

Setting Description
Automatically expand Trade Capture Reports Sets whether to expand collapsed entires.
Show tabs Sets whether to display tabs in the Blocktrader widget.
Set Execution Report columns Sets which columns to show in the Execution Report pane for a trade.

Order entry settings

Setting Description
Default order quantity Sets the default order quantity for each leg of the trade.
Reset quantity Sets whether to reset the order quantity to the default quantity or to keep the same quantity after submitting an order.
Confirm order before submission Sets whether to require confirmation before submitting an order.
