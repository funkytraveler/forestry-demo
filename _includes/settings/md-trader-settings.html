Display settings

Setting Description
Window Size Click to select the size of the widget.
Colors {% include settings/colors.html %}
Show price as net change {% include settings/show-price-as-net-change.html %}
Highlight midpoint of last recenter Sets whether to show the midpoint (white line) after the ladder is re-centered.
Show aggregator leg markets If trading an aggregated instrument and you enable this setting, MD Trader will open the corresponding legs of the instrument in separate MD Trader widgets.
Tick interval per row Sets the multiplier for the tick size at each price in the ladder. For example, if an instrument tick size is .25 and you set Tick interval per row to "2", each price in the ladder would tick by .50. The tick interval can consolidate the display of prices in the price ladder and display a greater range of prices while still showing the total quantity available. If you click the checkbox to enable this setting, the Tick Scale field appears in the MD Trader widget with the tick interval value that you entered.
Auto recenter grid (seconds) Sets how many seconds before MD Trader automatically re-centers the price ladder.
Recenter when market has moved (ticks) Sets whether to automatically recenter the trading ladder when the market moves a specified number of ticks since the last time it was re-centered.
Show number of orders Shows/hides the BCnt and ACnt columns. Check this option to show both columns. Uncheck to hide the columns. The BCnt column shows the number of orders comprising the total bid quantity at a price level. The ACnt column shows the number of orders comprising the total ask quantity at a price level.
Only show direct prices {% include settings/only-show-direct-prices.html %}
Show Liquidate button

Sets whether to show the Liquidate button that allows you to submit offsetting orders immediately to liquidate your position.

Note: This setting does not apply when MD Trader displays synthetic instruments, such as Autospreader or Aggregator instruments.
Show tabs Sets whether to display tabs at the bottom of the widget for the instruments used in the widget.
Reset order type to Limit after each order Sets whether to reset the Order Type drop-down to Limit automatically after an order with another order type is submitted.
