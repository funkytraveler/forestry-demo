When you dock an Order Ticket to a Market Grid, you create a single widget that automatically updates when you select different contracts.

To dock an Order Ticket:

  1. Open an Order Ticket from a Market Grid.

  2. From the right-click context menu for the Order Ticket, select Create widget group.

  3. Drag the Order Ticket to the top or bottom of the Market Grid.

  4. Release the mouse button to create the widget group.
