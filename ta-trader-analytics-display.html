The Trader Analytics display consists of the following:

  1. Time and Date selectors — Allows you to set a time interval for the calculations.
  2. Account selector — Allows you to calculate statistics for a single account, multiple accounts, or all accounts.
  3. Instruments selector — Allows you to calculate statistics for a single instrument, multiple instruments, or all instruments traded using the selected account(s).
  4. Calculate button — Calculates performance statistics for the selected account(s) and instrument(s).
  5. Performance Statistics — Rows and columns displaying performance statistics that are used for analyzing trader performance.
