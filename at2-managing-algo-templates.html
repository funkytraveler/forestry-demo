Creating an algo template

To create an algo template:

  1. Click Choose an algo, and select the desired algo from the list of deployed algos.
  2. Select Manage templates from the + Add row drop-down menu.

  3. In the Add template dialog, specify values for the desired algo instance parameters; then select + Add template from the Templates drop-down.

  4. In the Add template dialog, enter a name for the template; then click Save.

    The template is added to the drop-down.

Using an algo template

To use a template for an algo:

  1. Select an algo in the Autotrader widget.
  2. From the + Add row drop-down menu, select the template to use.

    An instance is added populated with the values defined in the template.

  3. Make any desired changes to the algo parameters.
  4. Click to start the algo instance with the selected template.

Modifying an algo template

To modify an algo template:

  1. Select Manage templates from the + Add row drop-down menu.

    An instance is added populated with the values defined in the template.

  2. In the template flyout:

    1. From the Templates drop-down, select the template you want to modify.
    2. Update the desired values for the selected template.
    3. Click the icon to save the changes.

Setting a default algo template

To set a default template for an algo in Algo Dashboard:

  1. Open an algo in Autotrader.
  2. Select Manage templates from the + Add row drop-down menu.
  3. Click the Templates drop-down menu.
  4. Click the icon for the template you want to make the default.

    Note: You can also click the icon to change the template name.

Deleting an algo template

To delete an algo template:

  1. Open an algo in Autotrader.
  2. Select Manage templates from the + Add row drop-down menu.
  3. Click the Templates drop-down menu.
  4. Click the icon for the template you want to delete.
