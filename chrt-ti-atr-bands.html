The ATR Bands indicator shows trends in price movements. Based on the average true range (ATR), the ATR bands are plotted around the ATR values to indicate the direction of movements in price.

{% include chrt-ti-images.html %}
  • {% include ti-period-blurb.html %}
  • Shift: Number of ATR multiples to draw the bands above and below the ATR.
  • {% include ti-field-blurb.html %}
  • Channel Fill: Whether to shade the area between the bands.
  • {% include ti-color-selectors-blurb.html %}
  • {% include ti-display-axis-blurb.html %}

Formula

ATR Bands take multiples of the ATR and plot them as bands above and below the ATR line.
