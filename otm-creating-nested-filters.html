You can create filters that combine single and group filters to create more complex filtering logic.

To create a nested filter:

  1. Click the Filter icon to open the Filter builder.

  2. Click Add rule

  3. Select the desired field you want to use in the filter; then select the desired criteria for the field.

  4. Click Add group.

  5. Select the desired field you want to use in the filter; then select the desired criteria for the field.

  6. Click Add rule and select the desired field you want to use in the filter; then select the desired criteria for the field.

  7. In the group area, click AND or OR as desired.

  8. Click Apply.

    The contents of the Options Trade Monitor widget updates to show only those trades that match the filter rule. In this case, the filter finds all trades for E2C or EW2 options whose trade quantity is greater than ten.
