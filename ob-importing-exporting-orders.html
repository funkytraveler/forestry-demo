CSV file format

tbd

  • Files must be saved as plain text format.
  • Filename must end in .csv.
  • Data fields must be comma delimited.
  • Spaces after commas (for readability) are allowed.

Sample .csv file

Version: 1.0
B/S, OrdQty, Exch, Contract, Price, TIF, Type, TrigPrc, DisQty, Account
B, 10, CME, ESM8, 21125.75, Day, Limit, , , ABC123
S, 25, CME, ZBM8, 162'24, GTC, StopLimit, 161'16, , ABC123
B, 100, CME, ESM8, 2109.00, Day, Iceberg, , 3, ABC123

File structure

  • Row 1 contains the user-defined version number
  • Row 2 contains the fields header, as follows:
    • The row must contain columns for all fields used by any of the orders.
    • The column names must exactly match the field names, including capitalization.
    • The fields can be arranged in any order.
  • Rows 3-n (up to 100) contain the values for a single order.
