Downloading TT Desktop

To download the TT Desktop installation package:

  • Method 1:

    1. In Microsoft Windows, log in to TT through a browser.

      The following is displayed.

    2. Click Download now.
    3. Save the TTSetup.msi file to the desired location on your file system.

  • Method 2:

    1. In Microsoft Windows, log in to TT through a browser.
    2. Open an existing workspace, or create a new one.
    3. From the workspace's File menu, select Download TT Desktop.
    4. Save the TTSetup.msi file to the desired location on your file system.

  • Method 3:

    1. Enter the trade.tt/desktop/download URL in your browser.

      Note: TT Desktop is not supported for TT Crypto (crypto.trade.tt).
    2. Specify the location to save the TTSetup.msifile, if prompted. If your browser is configured to automatically put downloaded files in a specific folder, this step will not be necessary.

Note: Alternatively, you can download the installer directly by entering the trade.tt/desktop/download URL in your browser.

Installing TT Desktop

Note: If you have a previous version of TT Desktop installed on your system, you should manually uninstall it before proceeding.

To install TT Desktop:

  1. Navigate to the location of the TTSetup.msi file you downloaded.
  2. Double-click the TTSetup.msi file.

  3. Click Next to install TT Desktop and progress through the installation dialogs. Then click Finish when the installation completes.

    After the installation process completes, Trading Technologies icon () is placed on your desktop.

Adding TT Desktop for a UAT environment

If your company has access to the UAT environment and your company administrator has given you permission to access the environment, you can also install TT Desktop for the UAT environment.

To add TT Desktop for UAT:

  1. Download the TTUatSetup.msi UAT installation package:
    • For methods 1 and 2 above, log into https://uat.trade.tt.
    • For method 3, enter uat.trade.tt/desktop/download in your browser.
  2. Double-click the TTUatSetup.msi file and following the installation instructions.
