Adding tabs

You can also add more tabs to the Market Grid to allow you to monitor different sets of instruments within the same widget.

To add a new tab:

  1. Click the + icon.

  2. Choose Select a product and either start typing in the field or use the explore feature to find the product you want to add.

  3. Click OK to add the new tab populated with the available contracts for the selected product.

Managing tabs

You can move, rename, copy, and remove tabs from the Market Grid.

To manage tabs:

  • To reorder the tabs, drag a tab to its new position. If you drag the tab away from the widget, a new Market Grid is added to the desktop populated with the instruments in the tab.
  • To rename a tab, select Rename tab from the tab's context menu, and specify the new name.
  • To restore a tab's name to its original value, select Restore tab from the context menu.
  • To copy a tab, select Copy tab from the tabs's context menu.
  • To remove a tab, select Remove tab from the tab's context menu.

Moving or copying instruments to a new tab

To move or copy an instrument to a new tab:

  1. Select one or more contracts you want to move or copy to a new tab.

  2. Right-click on a selected contract, and select either Copy to new tab or Move to new tab from the context menu.

  3. Enter a tab name, if desired, and click OK.

    A new tab is created with the selected contracts.
