After your Coinbase account is created at coinbase.com, sign in and create a Coinbase API key, API secret, and passphrase. These credentials are required in order to connect to Coinbase via the TT platform.

Multiple API keys and secrets can be created for a single Coinbase account.

Note: In order to test your algos or trade in the Coinbase test environment, you need to create a separate Coinbase account on public.sandbox.pro.coinbase.com for an individual account, or https://public.sandbox.prime.coinbase.com/ for an institutional account. After creating this account, log in to create an API key, API secret, and passphrase specifically for this environment.

To create a Coinbase API key:

  1. Log in to pro.coinbase.com if you are a TT Crypto user, or prime.coinbase.com if you are a trading firm user.

    Note: If you are creating an API key for the Coinbase "test" environment as a TT Crypto user, log in to public.sandbox.pro.coinbase.com/. If you are a trading firm (instituional) user, log in to https://public.sandbox.prime.coinbase.com/ to create an API key for the test (Simulation) environment.

  2. To create an API key, hover on the list icon on the right corner of the Exchange screen and select API.


  3. Only check the View and Trade Permissions checkboxes to set the API key permissions.

    Then click Create API Key to create and display the passphrase, new API key, and API secret. Copy or store these values on your computer or device.

    Note: This is the only time your API secret will display.


  4. Repeat this step to create additional API keys and secrets for the same Coinbase account if needed.

    5. Note: If you are a member of a trading firm who does not have administrator permissions in the Setup application on TT, then provide your risk administrator with your API key, secret, and passphrase. If you trade for yourself on TT and have administrator permissions in Setup, you can log in to the Setup application and create your trader ID for connecting to TT using your API key, secret, and passphrase.
