To edit a rule

  1. Click the edit icon next to a rule in the Autospreader Rules widget.

    The Edit Rule: (rule name) screen appears.

  2. Make the changes to the rule as needed as click Save.

    If prompted to confirm unsaved changes, click Save Changes.
