Many exchanges require identification of the trader responsible for submitting an order. To meet this requirement, TT supports the use of an Operator ID for exchanges where trader identification is required on manually and/or automatically submitted order actions. If configured, the Operator ID is sent to the exchange instead of the TT Username.

Exchanges Requiring Trader Identification

 

The following exchanges require trader identification on all submitted order actions:

  • CME
  • ICE
  • CFE
  • Euronext
  • MX
  • Eurex
  • BrokerTec
  • eSpeed
  • LSE
  • NASDQ OMX
  • NFX

The following table shows the Operator ID-to-Exchange API mapping done by TT Gateways.

 

Note: The value configured as the Operator ID must be an exchange-provided trader/user ID or a user-defined ID that is registered at the exchange.
