The following table shows the MD Trader local settings and workspace global preferences specific to trading crypto instruments.

Field Description
Quantity display decimal places Sets the preferred number of decimal digits to show for fractional quantities. You can set 0-8 decimal digits to display. If additional quantity is available, a "+" is shown to indicate the additional digits. Set this option in Preferences | Orders.
Price increment per row Consolidates prices in the price ladder. This setting adjusts the tick interval by the amount that you enter. Aggregate quantities are displayed at each price level. Set this option in Settings: MD Trader | Display.
Show only available prices Displays a floating price ladder by only showing price levels where there is available liquidity. Set this option in Settings: MD Trader | Display.
