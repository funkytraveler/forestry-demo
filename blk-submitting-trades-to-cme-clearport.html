• Enter a quantity and side, and click Units of Measure to set how the price is displayed and net sum is calculated.

    Trades with multiple legs show the net sum values for the bid and ask prices.

  • Set the transaction time for the trade.
  • In the Counterparty column, select a counterparty for the trade.

    Use the drop-down arrows in the counterparty fields to select the appropriate buy and sell accounts. Enter a few characters in the search field to quickly find the entry you need.

    A single click in the Side column toggles a leg to the opposite buy/sell direction.

    Optionally, check the Aggressor checkbox to set which counterparty is the aggressor in the transaction.

  • If needed, change the value in the Designated Broker field.

    This field is populated by default with your user alias defined in the Setup application. The Designated Broker field is the Clearport broker user ID assigned to you by CME when you registered with the exchange as a ClearPort user. Ideally, the ClearPort broker user ID should match your alias in Setup.

  • If the trade was negotiated off-exchange, click Voice brokered.
  • Click the TT account for routing selector, and select an account.

    The accounts listed are assigned to you in the Setup application and have valid connections for routing trades to CME ClearPort.

  • Optionally, populate the Text TT field.

    This field is not routed to the exchange, but the value remains on the order in the TT system.

  • Click Submit.

    The Confirmation screen appears.

    Note: Blocktrader prompts you to confirm orders by default. Uncheck Confirm order on the Blocktrader widget to disable order confirmations.

  • Confirm the order and click Submit to send the trade to CME ClearPort.
