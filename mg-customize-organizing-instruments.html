Organizing with label rows

You can add label rows to help you identify different sections of the rows in Market Grid. You might use this feature to provide visual cues for a Market Grid with many rows. For example, you could add label rows to add breaks based on years or quarters, similar to the following.

To add a label row:

  1. Right-click the row above which you want to add the label row, and select Insert label row.

  2. Enter the desired label text and press <Return>.

    The label row is added.

Note: You can also add a label row by selecting a row in the grid and typing the apostrophe (') key.

To rename a label row:

  1. Right-click on the label row text and select Edit label.

  2. Enter the new label name and press Enter.

To remove a label row, right-click on the row and select Remove row from the context menu.

Hiding inactive instruments

To enable Live Only mode and hide inactive instruments:

  1. Make the desired tab active.

    ADD IMAGE

  2. From the widget's context menu, select Enable Live Only.

    The inactive instruments are hidden and the tab and widget titles are updated to indicate that Live Mode is enabled, as shown.
