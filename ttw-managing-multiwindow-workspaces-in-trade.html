Using the workspace windows functionality, you can add and switch between multiple windows in a single workspace. After adding a window, you can also rename it, delete it, or set it as your default window.

Adding a window

To add a window:

  1. From the menu bar, click the Windows drop-down selector and click +Add window.

  2. Enter the name for the window, and click Save.

    The new window opens and is added to the Windows drop-down selector.

  3. Optionally, from the Windows drop-down selector, you can set the default window to open with the workspace, change the name of a window, and delete a window.

Switching between windows

To switch between windows:

  1. From the workspace title bar, click the Windows drop-down selector.

  2. Switch between windows using one of the following methods:

    • Select the desired window from the Windows drop-down selector.

    • Use the CtrL+window-number keyboard shortcut.

      For example, to switch to the Grains window in this case, you would type Ctrl+2.

Sending widgets to other windows

To send a widget to another window:

  1. Right-click in the widget's title bar to display its context menu.
  2. From the Send to submenu, select the desired target window.

  3. Switch to the selected window to see the moved widget.
