TT Order Type parameters

Parameters are described for the following TT Order Types:

  • TT Bracket order
  • TT Iceberg order
  • TT If-Touched order
  • TT OBV order
  • TT OCO order
  • TT Retry order
  • TT Stop order
  • TT Time Duration order
  • TT Time Sliced order
  • TT Timed order
  • TT Trailing Limit order
  • TT OCO Order Management Algo (OMA) order
