If you have access to Order Management Algos (OMAs), you can launch them for an order from the Floating Order Book. To launch an algo for an existing order:

  1. Click the arrow in the working order cell to open the Floating Order Book.

  2. Click to launch an OMA for a single order.

    Tip: You can select multiple orders and launch an OMA for each order by clicking the OMA button at the bottom of the grid. The OMA button displays the number of orders selected (e.g., OMA (2))

  3. Select the OMA to run.

  4. Enter the algo parameters and click .
