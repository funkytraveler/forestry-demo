Adding products to a Spread Matrix

  1. Click the + on a tab in the Spread Matrix to add another tab for a product.
  2. Select a tab from the Add Tab dialog that appears.

  3. Click Ok.

    The new tab for the single product or inter-product Spread Matrix appears at the bottom of the widget.

  4. Optionally, right-click the tab to rename it.
