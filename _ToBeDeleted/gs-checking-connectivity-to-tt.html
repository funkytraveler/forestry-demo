Using the File menu in your workspace, you can check your connectivity to the TT platform. For example, if private line access is enabled for your company, you can check which URL your browser is targeting for connecting to TT.

To check connectivity to TT:

  1. Open a workspace on the TT landing page.

  2. Click the File button on the titlebar.

  3. In the File menu, click About....

    The About... menu opens.

  4. The Connected to: line on the menu shows your connectivity to TT. In this example, the URL is a TNS private line to an edge server in the TT Chicago data center.
