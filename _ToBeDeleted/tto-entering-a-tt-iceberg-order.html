To enter a TT Iceberg order:

  1. In the the MD Trader® or Order Ticket widget, choose the contract to trade.
  2. Enter the desired order quantity and price for the order.
  3. Select TT Iceberg for the order type.
  4. Enter the desired Iceberg Details parameters.
  5. If you want to set trigger conditions for the order, check Trigger; then set the desired parameters.
  6. If you want to set start and end times for the TT order, set the desired parameters.
  7. Click OK to close the parameters dialog.
  8. Specify the TIF for the exchange-native child orders submitted by the TT Iceberg order.
  9. Click Buy or Sell to submit the order.
