The Williams Accumulation Distribution indicator was developed by Larry Williams. This indicator is calculated by finding the True Range High and True Range Low values for the day with:

True Range High = the higher value between yesterday's close and the day's high price.

True Range Low = the lowest value between yesterday's close and the day's low price.

. These values are the highest/lowest between yesterday's close and the day's opening prices.

Selecting the Williams Accumulation Distribution indicator allows you to set the Period (as number of minutes) and the color of the result.

Formula

If today's close is greater than yesterday's close:

then the Accumlation/Distribution will equal the today's close minus the True Range Low.

If today's close is less than yesterday's close:

then the Accumlation/Distribution will equal the today's close minus the True Range High.

The Williams' Accumulation/Distribution indicator is a cummulative total of today's Accumulation/Distribution and the Accumulation/Distribution from the previous day.
