--- widget: ow layout: task-tab title: Using order building mode AccordionID: ow-using-order-watcher-mode ---

The Conditional and OCO 2 TT public algos offer an option to use order building mode, which lets you dynamically build an OMA from existing or new orders. When watch mode is activated, the selected button and all MD Trader header panels are shaded yetllow to indicate that watch mode is active. You can then add or create orders to be managed by the OMA. New orders are added to MD Trader in a held state so they don't start working until the OMA is launched.

Also, custom action buttons that support watch mode can be added to any visible MD Trader widget. You do not need to add one to other MD Trader widgets whose orders you want to use in the OMA.

Adding OMA buttons to MD Trader for Order Watcher

Watch mode must be activated from a custom action button in an MD Trader widget. The custom action button must be assoiciated with a specific template for an OMA; you cannot use the default template for OMA algos you want to use in watch mode.

To add a custom action button for order building mode:

  1. From an MD Trader Order Entry panel context menu, select Edit custom action buttons.

    Note: You can add the button to any MD Trader widget.
  2. Select the OMA algo, then select a non-default template. You can also customize the button text, if desired.
  3. Click OK to add the button to the current MD Trader widget.

Selecting orders in watch mode

To select orders for an OMA in watch mode:

  1. Click the OMA button in MD Trader.

    The background color of the button and all MD Trader Header panels changes to yellow to indicate your are now in Order Watcher mode, and Order Watcher opens for the associated OMA algo.

  2. Either use shift-right-click to select an existing order or enter a new order in an MD Trader widget.

    The selected or new order is added to order builder.

    1. Shift-right-click the working order to add it to the algo order.
    2. The selected order appears in the order builder.
  3. Either use shift-right-click to select an existing order or enter a new order in an MD Trader widget.

  4. In the order builder, click Launch algo.

    You can also enable select Auto-launch to launch the algo automatically when the algo's required number of orders is added.

    The new algo parent order, along with its child orders, are added to the Order Book.
