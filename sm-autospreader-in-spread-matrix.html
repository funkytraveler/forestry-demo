Autospreader functionality for creating spreads in Spread Matrix

Using functionality from the Autospreader widget in the Trade application, you can dynamically create inter-product synthetic spreads in Spread Matrix by:

  • Entering an order from a floating MD Trader or Order Ticket for an inter-product Spread Matrix
  • Using the Edit spread... context menu option for an inter-product Spread Matrix

Inter-product spreads created from the Spread Matrix can be managed in the Autospreader widget, and they can be launched from the Market Explorer and traded like exchanged-defined spread instruments.

After entering the order, Spread Matrix creates a synthetic inter-product spread instrument that can be managed and edited in the Autospreader® Manager.

The Edit Spread... setting is only available for synthetic inter-product spreads or spreads created via the Custom tab in Spread Matrix. After creating and saving the spread, it appears in the Autospreader® Manager.
