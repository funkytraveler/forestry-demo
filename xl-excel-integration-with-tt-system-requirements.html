Excel integration is currently supported on Windows systems with the following software:

  • Microsoft Windows 7 with Service Pack 1, or higher

  • Microsoft Excel 2013, or higher, running locally

  • Microsoft .NET 4.6 framework

    Note: Installing the TTExcelIntegration add-in will automatically install .NET 4.6, if necessary

  • Browsers: Google Chrome, Internet Explorer, or Microsoft Edge
