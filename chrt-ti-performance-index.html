The Performance Index indicator is used to compare the price trend of one instrument with the performnance of a benchmark index.

Note: The Performance Index is valid only when comparing multiple instruments using either Comparisons or Series. When more that two instruments are used, each correlation indicator is compared to the chart's first instrument.

Formula

\[ PI = \frac {Instr\ Close}{Index\ Close} \times \frac {Index \ MA}{Instr \ MA} \]

where MA is the moving average.
