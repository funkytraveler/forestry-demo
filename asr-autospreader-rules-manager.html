Autospreader Rules Manager

After you open the Autospreader Rules widget, the Autospreader Rules Manager opens and displays the list of rules you've created, as well as the default TT rules that are available as part of the TT platform. You can create and edit your rules using the Autospreader Rules widget:

The Autospreader Rules Manager includes the following functions:

  • Create — Opens an Autospreader Rules configuration dialog box for creating a new rule.
  • Edit (icon) — Opens an Autospreader Rules configuration dialog box for modifying an existing rule
  • Copy — Allows you to select and copy an existing rule. The rule can then be modified and renamed as needed.
  • Delete — Allows you to select and delete a rule.
  • Filter list — Allows you to enter a rule name, description, or date and filter the list of rules.
