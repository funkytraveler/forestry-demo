The following uses cases show how you can use the Options Chain to create strategies:

  1. Creating a butterfly strategy
  2. Creating a strip strategy
  3. Creating a custom straddle strategy

Example: creating a butterfly strategy

If you know the structure of a particular type of strategy, you can add the legs with the appropriate ratios by clicking them in the correct order. In this example, you can use the following procedure to create a butterfly spread:

  1. In the Options Chain widget, click Fly to enter Spread Building mode.

  2. Click a Bid or Ask price to specify the first leg of the strategy.

  3. Click the price for either the second or third leg of the butterfly strategy.

    As you hover over a price, the butterfly strategy automatically identifies the missing leg of the spread. In this case, you specify the second leg, so the third leg is automatically highlighted.

    The Strategy Creation widget opens with the legs already added.

  4. If you want to cover your position in the options with a position in the underlying futures contract, click + to add the futures instrument.

  5. In the Strategy Creation widget, make any desired adjustments and click Create to submit your options strategy to the exchange.

Example: creating a strip strategy

In this example, you can use the following procedure to create a strip spread that sells one call options contract for each of the expiration months from Dec2016 to Jun2017:

  1. In the Options Chain widget, click Strip to enter Spread Building mode.

  2. Click a Bid or Ask price to specify the first leg of the strategy.

  3. In the drop-down menu, select the end of the desired range of expiration months.

    The Strategy Creation widget opens with the legs already added. The strategy is also seeded with the front month futures leg.

  4. If you want to cover your position in the options with a position in the underlying futures contract, click + to add the futures instrument.

  5. In the Strategy Creation widget, make any desired adjustments and click Create to submit your options strategy to the exchange.

Example: creating custom straddle strategy

By clicking on the Strike column of an option, you can easily create a straddle strategy. In this example, you can use the following procedure to create a Buy or Sell straddle strategy at the 98 strike price:

  1. In the Options Chain widget, click Custom.

  2. Left-click a price in the Strike column for a Buy straddle, or right-click a price to create a Sell straddle.

    The widget automatically selects a Put and a Call option at that price level.

  3. Click Submit to create the corresponding straddle strategy.

    Note: If you want to cover your position in the options with a position in the underlying futures contract, click + to add the futures instrument. In this case, you added the underlying instrument to the Sell strategy.

  4. In the Strategy Creation widget, make any desired adjustments and click Create to submit your options strategy to the exchange.

Example: creating basic straddle strategy

By clicking on the Strike column of an option, you can easily create a straddle strategy. In this example, you can use the following procedure to create a Buy or Sell straddle strategy at the 44.50 strike price:

  1. In the Options Chain widget, press and hold the Ctrl key to enter Spread Building mode.

  2. Left-click a price in the Strike column for a Buy straddle, or right-click a price to create a Sell straddle.

    The widget automatically selects a Put and a Call option at that price level.

  3. Release the Ctrl key to create the corresponding straddle strategy.

  4. In the Strategy Creation widget, make any desired adjustments and click Create to submit your options strategy to the exchange.
