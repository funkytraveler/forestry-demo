To create a rule:

  1. Click Create Rule in the AutospreaderÂ® Rules widget.

    You can also open the Autospreader Rules Create Rule screen in Autospreader by clicking the "+" in the Rules field.

  2. Enter a rule name and description.
  3. Select a rule type.
  4. Select a condition type.
  5. Optionally, click the Custom Variable tab to define your own variables for the condition or action logic.
  6. Enter the condition logic.
  7. Click Add an Action... to select an action based on your rule type.
  8. Enter the action logic.

  9. Click Save.

    The rule appears in the Autospreader Rules widget's list of rules, and can be selected in the Autospreader widget when configuring a spread.
