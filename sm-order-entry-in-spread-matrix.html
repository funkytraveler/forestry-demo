Floating order entry in Spread Matrix

Using the Spread Matrix and a floating order entry window, you can quickly enter orders for all outrights and calendar spreads within a single window for active spread trading. To trade the spread, left click on any of the four cells to open your floating order entry window.

Note: The cells containing spread contract names are highlighted to indicate which contracts you are spread trading.

Use the Order Entry settings in Preferences to set which floating order entry to use: Order Ticket or MD Trader.

You can also right-click in the market data cells for a spread to view the context menu for the calendar spread. In the context menu, you can select Open to open an MD Trader or Order Ticket to enter an order for the selected calendar spread.

Managing orders in Spread Matrix

After submitting an order from the Spread Matrix, you can manage the order using the Floating Order Book.

Based on the Preferences | Orders | Launch floating order book using: option you select, you can open the Floating Order Book from the Bid Quantity or Ask Quantity cell for an instrument in the matrix. Once opened, use the Floating Order Book to modify or cancel working orders that you submitted using the Spread Matrix or other order entry widget.

Managing your position in Spread Matrix

Your position displays below each of the outrights in the matrix, making it easy to see your position for all expirations of the product. Blue indicates that you are long, red indicates that you are short, and gray indicates that you are flat.
