Spread Matrix tabs

The Spread Matrix widget can be opened from the widgets menu. When opened, the Spread Matrix displays the following tabs on the Add Tab screen:

  • Exchange Listed — Opens a Spread Matrix for a single product and displaying market data for exchange-traded calendar spreads.
  • Autospreader — Opens an inter-product Spread Matrix to display synthetic spreads that can be created in the matrix and managed in Autospreader®.
  • Custom — Opens an inter-product Spread Matrix that you can customize, and displays synthetic spreads that can be created in the matrix and managed in Autospreader®.

Exchange Listed tab

On the Exchange-listed tab, click Search for a product... to show the search box and market explorer to launch Spread Matrix for a single product exchange-traded spread.

Autospreader tab

You can use Spread Matrix to display market data for multiple instruments of two different products and all of the associated spreads between those instruments. You can use the Autospreader tab to use previously defined two-legged, Autospreader® spreads to populate a Spread Matrix, or you can use the Custom tab if the desired synthetic spread does not already exist.

Note: The Autospreader tab only displays synthetic inter-product spreads. If there are no inter-product spreads created in Autospreader®, this dropdown will be empty.

Custom tab

To populate a Spread Matrix with two different products when there is not an existing Autospreader spread comprised of legs of the two different products, click the Custom tab. This tab provides basic Autospreader® functionality for configuring a two-legged, synthetic inter-product spread.
