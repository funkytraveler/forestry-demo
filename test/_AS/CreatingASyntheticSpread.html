Before you begin, open the Autospreader widget.

  1. Click Create in the Autospreader widget.

  2. Set the following spread definition parameters:

    • Name
    • Color
    • Spread formula
    • Custom spread type (optional)
  3. Select a contract for each leg of the spread. Click + to add legs.

  4. For each leg, configure the following settings:

    • Ratio
    • Multipler

  5. For each leg, configure the following quoting and/or hedging order execution settings as needed:

    • Pay Up Ticks
    • Active Quoting
    • Minimum Lean Quantity
    • Queue Holder
    • Reload
  6. Optionally, click Rules + to add rules for quoting, pre-hedge, and post-hedge orders.
  7. Optionally, override the calculated tick size for the synthetic spread.
  8. In the Spread Preview pane, view the synthetic market for the spread.
  9. Click Save
