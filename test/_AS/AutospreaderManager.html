When you open the Autospreader widget from the Widgets menu, the Autospreader Manager opens to show you the list of spreads you've created and those that have been shared with you. You can launch, create, edit, and manage (copy, share, delete) all of your spreads from this widget.

Update links

Autospreader functions

The Autospreader Manager includes the following functions:

  • Create — Opens an Autospreader configuration dialog box for creating a new spread.
  • Launch — Launches a spread in your workspace using one of the selected options.
  • Edit (icon) — Opens an Autospreader configuration dialog box for modifying an existing spread
  • Copy — Allows you to select and copy an existing spread. The spread can then be modified and renamed as needed.
  • Delete — Allows you to select and delete a spread.
  • Filter list — Allows you to enter a spread name, status, owner, or date and filter the list of spreads.
